Design Fix Requirements
1. Typography Scale Reduction
Title text ("Review: Forex Fu...", "Major Update:...", "XAUUSD M1..."): Reduce from current large size to 14px-16px (max 16px) to match Top Sellers' compact style.​

Meta text ("by Unknown", "5,235", "3", "about 8 hours ago"): Use 12px-13px for secondary info, matching Top Sellers' smaller meta font.​

Category tags ("ea-reviews", "news-updates", "strategy-discussion"): Keep at 11px-12px with tight padding.​

2. Compact Card Layout
Reduce vertical padding between elements: Use 8px-12px spacing inside each card instead of current scattered layout.​

Tighter line-height: Set line-height to 1.4-1.5 for title text (currently looks like 1.8+).​​

Unified card height: Make all cards equal height with consistent internal spacing, matching Top Sellers' uniform appearance.​

3. Visual Hierarchy Alignment
Rank badge (1, 2, 3): Keep current size but ensure consistent alignment with text baseline.​

"Hot" badge: Reduce size slightly, position consistently on right edge with 4px-6px margin from card edge.​

Icon/view/comment counters: Use single-line horizontal layout with icons at 12px-14px, text at 12px, separated by 8px gaps.​​

4. Text Alignment & Spacing
Left-align all text with consistent 12px-16px left margin from rank badge.​

Remove excessive whitespace: Collapse scattered text into compact blocks with 4px vertical gaps between title → meta → stats → category.​

Consistent right padding: Ensure "Hot" badge doesn't push text; maintain 40px right padding for badge space.​
